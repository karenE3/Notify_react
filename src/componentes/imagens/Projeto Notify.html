Variáveis e Estruturas de dados

Para criação de variaveis na linguagem python não precisamos definir um tipo, logo chamamos de uma linguagem não tipada.
Veja alguns exemplos abaíxo:

variável de texto (string)
nome = 'Gabriel Roger'

variável inteira (int)
idade = 23

variável decimal (float)
salário = 2999.43

variável booleana (boolean)
empregado_atualmente = True


Para estas variáveis, poderíamos definir outros valores, como por exemplo no tipo booleana (True ou False).
Entendemos que variável é armazenar um dado em um espaço na memória, seu dado pode mudar conforme a necessidade das regras lógicas do sistema,
e o sinal (=) significa atribuição.
No python além de armazenarmos informações dos tipos comentados, podemos estruturar os dados, ou seja organiza-los para que fique mais fácil
de trata-los ou até mesmo desenvolvermos nossa lógica de programação.
Algumas estruturas de dados que usaremos são listas, dicionários e tuplas.

listas:

Para definição de uma lista utilizamos [] "colchetes".

Criação de uma lista vazia:
lista = []

observação: Perceba que no momento em que definimos uma lista, criamos uma variável que irá guardar um conjunto de dados.

usamos para organizar um conjunto de dados, por exemplo:
salada_de_frutas = ['morango', 'maçã', 'banana', 'leite moça']

Nas listas podemos guardar diferentes tipos de dados como:
tipo_de_dados_diferentes = [True, 18.00, 'Gabriel', 28, False]

Todas as vezes que criamos uma lista possuem funções internas do python em que nos auxiliam no seu trabalho.
Nos exemplos abaixo estamos utilizando listas com suas funções internas:

# Criação de uma lista vazia:

lista = []

# Adicionando um ou mais elementos a esta lista:

lista.append('item1')
lista.append('item2')

# Quando realizar um print na lista que antes era vazia, seu resultado será este:

print(lista)

#resultado

['item1', 'item2']

Toda lista possui suas posições em que chamamos de índices, estes índices são acessados da seguinte forma:
print(lista[0])

#resultado:
item1

Neste caso o índice 0 é o primeiro elemento desta lista, e o índice 1 é o segundo elemento.

Projeto Notify

Lucas de Melo 12 de jul Editado às 12 de jul
1 comentário da turma
Vamos criar nosso primeiro site profissional? <3
NOTIFY

O intuito do Projeto Notify é que repliquem o modelo do site que está em PSD utilizando todos as ferramentas que aprendemos com HTML e CSS entre os dias 01 de julho e 11 de julho para consolidarmos o conhecimento.

- O tempo de execução será de 4 horas;
- O trabalho deverá ser em dupla ou trio a serem escolhidos pelos professores;
- Todos os grupos deverão postar nesta tarefa o link do repositório do GITHUB, em uma conta pessoal;
- A atividade é obrigatória à todos os alunos.

Em anexo, há um pacote com as imagens necessárias para a criação do site.

Deverão utilizar as ferramentas:
- HTML;
- CSS ( Display Flex, CSS Grid, BoxModel );

Para fazer o donwload da imagem:
- Abra a imagem
- Clique nas bolinhas no canto superior direito e clique em OPEN IN NEW WINDOW
- Clique no símbolo de download no canto superior direito
